# Team Arion Website (Early Version)

This repository preserves an early version of the Team Arion website for archival and historical purposes.
It is no longer maintained.

Tech stack: static HTML, CSS, and JavaScript.

## Structure

- `index.html` is the interactive display entry point.
- `assets/css/styles.css` contains consolidated styles.
- `assets/js/scripts.js` contains consolidated scripts.
- `assets/img/` contains image assets.
