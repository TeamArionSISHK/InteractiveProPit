This repository contains the legacy interactive touchscreen web interface used in Team Arion’s Professional Class competition Pit Display. It served as an on-site interactive system for presenting team information, engineering highlights, and visual materials during competition events.

This project is preserved for archival and historical reference and is no longer actively maintained.

## Purpose
The system was developed as part of Team Arion’s competition infrastructure, supporting:
- Interactive 3D model viewer using Autodesk Fusion's web embed. 
- Portfolios showcase
- QR codes and etc.

## Structure
- `index.html` is the interactive display entry point.
- `assets/css/styles.css` contains consolidated styles.
- `assets/js/scripts.js` contains consolidated scripts.
- `assets/img/` contains image assets.

## Tech Stack
Static HTML, CSS, and JavaScript.
